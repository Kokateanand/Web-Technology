## ✨ Features

- **👥 Student Directory:** A complete registry showcasing individual student profile submissions.
- **💻 Source Code Access:** Standardized links directing to Google Drive hosting student codes.
- **⚡ Live Previews:** Fast preview endpoints referencing active server directories for student projects.
- **📱 Responsive Design:** Sleek modern styling powered by Bootstrap 5 and customized dark mode guidelines.

This repository functions as a static showcase of educational submissions.
